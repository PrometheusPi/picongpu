PICMI
=====

PIConGPU supports `PICMI <https://picmi-standard.github.io/>`_ to create simulation setups from Python.

.. warning::

   The PICMI support is still **experimental**.
   You are very much encouraged to give feedback on your workflow, this documentation, the PICMI definition and everything related.
